# NGO Growth Action Planner 🚀📊

An interactive, intelligence-powered platform for Non-Governmental Organizations (NGOs) and social organizations to analyze current practices, obtain custom-tailored strategic advice, curate action steps, test readiness, and export comprehensive growth plans.

---

## 🌟 Key Features

*   **Dynamic Strategic Formulator:** Inputs target sector, physical coordinates, budget bracket, operational scale, and main bottlenecks to create a tailored program roadmap.
*   **Operational Readiness Diagnostic Scorecard:** Self-assessment checklist tracking critical parameters—including micro-video storytelling, tasks discretization, unit-based donation math, fast mobile responsiveness, and public transparency indexes history.
*   **5 Pinned Strategic Pillars:** Select, filter, and organize exactly 5 cohesive pillars across Outreach, Volunteer Engagement, Fundraising, and Digital Brand Trust.
*   **Organizer Live Draft Notes:** Add direct action assignments, delegate names, and custom scheduling timelines directly to each strategic milestone.
*   **Two-Way Professional Exporter:**
    *   **Academic Page Design:** A meticulously designed executive blueprint containing metadata panels, executive statements, specific workflows, and validation signature zones. Compatible with modern system print templates (Print to PDF).
    *   **Digital Clipboard Copy:** Plain text converter structures key results for immediate upload to document tools or local drives.
*   **Task-Outreach Engagement Toolkit:** Built-in copy utilities for professional sharing, automated hashtags (`#InAmigosFoundation #SocialImpact #NonprofitGrowth`), and direct portals to satisfy outreach campaign updates.

---

## 🛠️ Architecture & Technologies

*   **Frontend Ecosystem:** React 19, TypeScript, Tailwind CSS, Lucide Icons, and Space Grotesk elegant display layout.
*   **Backend Server:** Node.js Express service for streamlined local execution and dynamic intelligence routing.
*   **Build Optimization:** Bundled via Esbuild to generate high-performance, single-file CommonJS formats to optimize cold-starts in containerized runtimes.

## 🚀 Running the Project Locally

### Prerequisites
*   Node.js (v18+ recommended)
*   npm or yarn

### 1. Installation
Install the required packages:
```bash
npm install
```

### 2. Configuration
Create a `.env` file in the root directory and define the required parameters:
```env
# Optional secret keys for customized intelligence routing.
# Defaults to a high-impact, curated baseline repository when empty.
GEMINI_API_KEY=
APP_URL=http://localhost:3000
```

### 3. Start Development Server
Launch the unified TypeScript server:
```bash
npm run dev
```
The application will be accessible at [http://localhost:3000](http://localhost:3000).

### 4. Production Build & Execution
Compile static client assets and server bundles:
```bash
npm run build
npm start
```

## 📊 Deployment Strategy
The platform is designed to be fully compatible with cloud container ingress routing, automatically binding to port `3000` and host `0.0.0.0`. 
To export the printable master plan as a file:
1. Complete the profile form.
2. Select 5 strategies from the repository.
3. Click **Print / PDF** to generate an executive-style blueprint.
4. Save the PDF and upload it to your remote shared folders.
